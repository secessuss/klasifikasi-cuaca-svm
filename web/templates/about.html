Tentang Proyek Klasifikasi Cuaca

Tujuan Proyek

Proyek ini bertujuan untuk membuat aplikasi web yang dapat mengklasifikasikan gambar ke dalam empat kategori cuaca: Berawan (Cloudy), Hujan (Rain), Cerah (Shine), dan Matahari Terbit (Sunrise). Aplikasi ini merupakan implementasi dari model machine learning klasik yang telah dilatih sebelumnya.

Dataset

Model dilatih menggunakan dataset "Multi-class Weather Dataset" yang tersedia di Kaggle. Dataset ini berisi ribuan gambar yang sudah dikelompokkan ke dalam empat kelas cuaca yang disebutkan di atas.

Metodologi Machine Learning

Proses pelatihan model menggunakan pendekatan machine learning klasik dengan langkah-langkah sebagai berikut:

  • Ekstraksi Fitur: Mengubah gambar menjadi data numerik menggunakan kombinasi metode HOG (Histogram of Oriented Gradients) untuk bentuk, Color Histogram untuk warna, dan LBP (Local Binary Patterns) untuk tekstur.
  • Reduksi Dimensi: Menerapkan Principal Component Analysis (PCA) untuk mengurangi jumlah fitur, mengatasi overfitting, dan meningkatkan efisiensi model.
  • Model Klasifikasi: Menggunakan algoritma Support Vector Machine (SVM) dengan kernel RBF yang telah dioptimalkan melalui hyperparameter tuning.
  • Hasil: Model akhir mencapai akurasi sekitar 95% pada data uji, menunjukkan kemampuan yang baik dalam membedakan keempat kelas cuaca.

Kredit

Aplikasi ini dibuat sebagai bagian dari tugas kuliah untuk mendemonstrasikan penerapan machine learning dalam aplikasi dunia nyata.
